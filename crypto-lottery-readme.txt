# CryptoLotería 🎲💰

## Descripción
CryptoLotería es una aplicación web de lotería de criptomonedas para Argentina y Uruguay, donde los usuarios pueden ganar criptomonedas viendo anuncios y participando en sorteos.

## Características
- Dos turnos de lotería diarios (mañana y noche)
- Ganar criptomonedas viendo anuncios de YouTube
- Retiro de fondos a billeteras virtuales
- Diseño responsive
- Totalmente gratuito

## Requisitos Previos
- Node.js (v14 o superior)
- npm o yarn

## Instalación
1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/cryptoloteria.git
cd cryptoloteria
```

2. Instalar dependencias
```bash
npm install
# o
yarn install
```

3. Iniciar servidor de desarrollo
```bash
npm start
# o 
yarn start
```
